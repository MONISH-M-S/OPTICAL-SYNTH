Layer 01 · Physics
Photo-Resistive
Transduction

The LDR is made from a high-resistance semiconductor. When photons from a phone torch strike the surface, they transfer enough energy to free electrons inside the material.

More free electrons → greater conductivity → lower resistance. This creates the inverse relationship: light ↑ → Ω ↓

Because the Arduino cannot measure resistance directly, a 10kΩ resistor forms a Voltage Divider, converting changing resistance into a 0V – 5V signal.

Layer 02 · Logic
Digital
Processing

The voltage enters Analog Pin A0. The internal ADC samples it and outputs an integer in 0 – 1023.

The map() function translates that value to a musical frequency, e.g. a raw value of 800 becomes ~1500 Hz.

tone() uses the Arduino's internal clock to pulse Pin D8 at exactly that frequency — a square wave output.

Layer 03 · Acoustics
Electro-Mechanical
Conversion

An 8Ω speaker would draw dangerously high current from the Arduino pin. A 220Ω resistor acts as a buffer, keeping current within safe limits.

Electrical pulses flow through the voice coil, creating a fluctuating magnetic field that pushes and pulls against a permanent magnet.

The resulting cone movement compresses air into longitudinal pressure waves — the sound you hear.

optical_synth.ino Arduino C++ 
// ── Pin Definitions ──────────────────────────────────
const int speakerPin = 8;
const int pitchLDR   = A0;  // Controls Pitch
const int modLDR     = A1;  // Controls Tempo/Stutter

void setup() {
  pinMode(speakerPin, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  // 1. Read raw ADC values (0–1023)
  int pitchRaw = analogRead(pitchLDR);
  int modRaw   = analogRead(modLDR);

  // 2. Map to audible ranges
  int frequency  = map(pitchRaw, 200, 950, 150, 2500);
  int pulseSpeed = map(modRaw,   200, 950,  10,  200);

  // 3. Constrain to safe values
  frequency  = constrain(frequency,  150, 2500);
  pulseSpeed = constrain(pulseSpeed,  10,  200);

  // 4. Stutter: tone ON → delay → tone OFF → delay
  tone(speakerPin, frequency);
  delay(pulseSpeed);
  noTone(speakerPin);
  delay(pulseSpeed);
}
